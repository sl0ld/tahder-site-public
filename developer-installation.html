Developer Installation

تثبيت نسخة المطور من إضافة تحضيري

هذه الصفحة مخصصة للمطورين أو فريق الاختبار فقط. المستخدم العادي لا يحتاج إلى Developer Mode أو Load unpacked.

العودة لطريقة التثبيت العادية
تنبيه مهم

استخدم هذه الطريقة فقط أثناء التطوير أو اختبار النسخة المحلية قبل رفع الإضافة لمتجر Chrome.

  1. 1
    افتح صفحة الإضافات اكتب في المتصفح: chrome://extensions
  2. 2
    فعّل Developer Mode من أعلى صفحة الإضافات، فعّل خيار وضع المطور.
  3. 3
    اختر Load unpacked اضغط الزر ثم اختر مجلد إضافة تحضيري من جهازك.
  4. 4
    اختبر داخل منصة مدرستي افتح صفحة تحضير مناسبة وتأكد أن اللوحة لا تظهر في صفحات تسجيل الدخول أو الصفحات العامة.
